## ğŸ“Š Project Overview

This project implements an end-to-end Machine Learning pipeline to predict residential real estate prices based on 79 explanatory variables. It covers every step of the Data Science lifecycle, from raw data cleaning to advanced regression modeling.

### Key Features
- **Exploratory Data Analysis (EDA):** Deep dive into correlation matrices and outlier detection.
- **Feature Engineering:** Log-transformation of skewed target variables, handling missing values, and one-hot encoding.
- **Model Selection:** Comparison of Linear Regression, Random Forest, and Gradient Boosting Regressor.
- **Evaluation:** Optimized for RMSE (Root Mean Squared Error).
